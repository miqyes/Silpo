# Silpo
# 🛒 Silpo Online Order System

![Status](https://img.shields.io/badge/Status-Completed-success) ![C++](https://img.shields.io/badge/Language-C++-blue)

A console-based order management system simulating the "Silpo" supermarket online shopping experience. This project demonstrates core **OOP** principles, **STL containers**, and **File I/O** operations.

---

## 🌟 Key Features
* **Cart Management:** Add, remove, or update product quantities.
* **Smart Discount Engine:**
    * 🏷️ **"1+1=3" Deal:** Every third identical item is free.
    * 🥛 **Item of the Day:** Automatic **30% discount** on Milk.
    * 🎫 **Promo Codes:** Support for codes like `RACCOON_HERO`.
* **Order Customization:** Online/Upon Receipt payment and custom delivery dates.
* **Data Persistence:** Automatically saves/restores cart from `last_order.txt`.

---

## 🛠 Technical Stack
* **OOP Architecture:** Encapsulation, constructors, and getter/setter methods.
* **STL Vectors:** Dynamic management for the shopping cart.
* **File I/O:** Persistent storage for order history.
* **Input Manipulation:** Formatted currency output (2 decimal places).

---

## 📁 Project Structure
* `Silpo order.h` — Header file with class definitions.
* `silpo.cpp` — Implementation of the logic.
* `main.cpp` — Interactive console menu.
